  - 1.(cd xxx)cd到项目文件夹
  - 2.(touch README.md)假如在github创建仓库的时候没有添加README文件，先创建文件
  - 3.(git init)git初始化项目
  - 4.(git add *)添加所有文件
  - 5.(git commit -m "注释")提交注释
  - 6.(git remote add origin git@github.com:xxxxxxx/YPhotoAutoLayout.git) 连接远程仓库
  - 7.(git push -u origin master)提交到远程仓库，刷新网页，看到添加成功
